# Changelog

- Repositioned CommerceLens as a private commercial product.
- Added first-class account, project, and member records.
- Added admin account/project/member APIs.
- Added a private operator dashboard for accounts, jobs, runs, API keys, and usage.

- Added production environment template.
- Added proprietary license posture.
- Added centralized package/API version.
- Added `/ready` readiness endpoint for hosted deployments.
- Added configurable hosted fetch user agent and timeout environment variables.
- Added production, commercialization, and security documentation.
